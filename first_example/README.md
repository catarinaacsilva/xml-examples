# First Example - xml

- Added 2 files to test:

    - mect.xml
    - curso.xsd

- When validation fails, it is created a log file with errors.

## Requirements

- Python 3

- lxml

## Run

1. Create a venv:

    `python3 -m venv venv`

    `source venv/bin/activate`

2. Install lxml

    `sudo pip install lxml==4.4.1`

3. Run xml_proc.py

    `python3 xml_proc.py`
